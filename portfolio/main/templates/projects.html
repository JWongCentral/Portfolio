Parallel Minesweeper Solver

Description:

The parallel Minesweeper solver program utilizes the power of C++ to solve Minesweeper boards concurrently. By dividing the task among multiple threads or processes and using AI forward checking procedure (AC3) and backward checking, to maximizes computational efficiency, effectively revealing safe cells and flagging mines in the game, resulting in faster completion times and improved performance.

Packages/Concepts used:

  • - OpenMP a parallelization package for shared memory processes

  • - Arc Consistency 3 - as a forward/backward checking procedure

  • - Using a heuristics such as MCV (Most Constrained Variable) to improve performance and speed
